Boutique du Sommeil

Projet de création de boutique en ligne.

Vous pouvez librement tester ce projet.

Le bouton "Payer" ouvrira simplement une pop-up avec écrit "Merci".

Projet réalisé en Angular 9.

Il est hébergé par Firebase Hosting (ici).

La sauvegarde du panier s'effectue dans le "local storage" du navigateur.

Librairies utilisées pour la mise en forme : Bootstrap 4, Material Design Angular et ngx-slider.

Temps estimé passé sur ce projet : 18h.

Principal challenge : transmettre les modifications du panier à tous les composants.

Pour cela, j'ai utilisé mis en place des events et des observables proposés par Angular.

Code disponible sur GitHub : https://github.com/romainGeffrault/Boutique-du-sommeil.
